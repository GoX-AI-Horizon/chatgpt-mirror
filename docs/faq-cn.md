# 常见问题

## 1. 为什么首页会提示 403

```
正在验证您是否是真人。这可能需要几秒钟时间
验证所用的时间比预期时间长。如果问题仍然存在，请检查您的 Intemnet 连接并刷新页面
```

- 更换服务器 IP 解决
- 使用代理
- 使用 warp

## 2. 为什么向 ChatGPT 提问时无法解析 URL 链接和文件，而官网可以

这是由于 IP 受到 ChatGPT 功能限制，可以通过更换服务器 IP 解决，或者切换本地代理。
